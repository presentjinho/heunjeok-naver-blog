# 기여하기

## 개발 환경

Node.js 22 이상을 사용합니다. 외부 패키지 설치 없이 실행할 수 있습니다.

```text
npm test
npm run security:scan
npm start
```

## 변경 원칙

- 네이버 블로그 전용 범위를 유지합니다.
- 사용자 경험·가격·효과·최신 정보를 새로 만들어 내지 않습니다.
- 비밀번호·쿠키·OAuth 토큰·API 키를 브라우저 저장소나 로그에 넣지 않습니다.
- 사용자 입력을 서버에 새로 저장하는 기능은 보존 기간·삭제 기능·동의를 함께 구현해야 합니다.
- 기능 변경에는 충돌·회귀·보안 테스트를 추가합니다.
- 자동 발행·댓글 자동 등록·브라우저 DOM 조작은 제안하지 않습니다.

보안 취약점은 공개 Issue 대신 [SECURITY.md](./SECURITY.md)의 신고 절차를 사용해 주세요.
