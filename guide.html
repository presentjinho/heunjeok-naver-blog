흔적 사용 방법

흔적은 네이버에 자동으로 로그인하거나 글을 발행하지 않습니다. 내 경험으로 초안을 준비하고 일반 텍스트로 복사하는 로컬 작업실입니다.

가장 빠른 순서

  1. 분야와 주제를 하나 고릅니다.
  2. 핵심 경험 세 칸에 각각 다섯 글자 이상 적습니다. 또는 추가 메모에 실제 경험을 30자 이상 적습니다.
  3. 초안을 만들고 숫자·가격·광고 관계를 확인합니다.
  4. 제목·본문·태그를 복사해 네이버 글쓰기에 붙여넣습니다.
  5. 사진·장소·공개 범위는 네이버에서 마지막으로 확인합니다.

저장과 기기 이전

답변과 초안은 현재 브라우저에만 저장됩니다. 다른 휴대폰이나 PC로 옮기려면 설정 영역의 ‘내 데이터 백업’으로 JSON 파일을 만든 뒤 새 기기에서 가져오세요. 사진은 백업 파일에 포함되지 않습니다.

복사가 막힐 때

자동 복사가 실패하면 수동 복사창이 열립니다. 전체 선택을 누른 뒤 길게 눌러 복사하세요. 경험을 수정한 뒤에는 오래된 초안을 다시 만들어야 복사할 수 있습니다.

네이버 공식 서비스가 아니며 검색 노출·순위·수익을 보장하지 않습니다.
