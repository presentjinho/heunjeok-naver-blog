# 개인정보 처리 기준

이 문서는 공개 배포 전 운영자가 실제 회사명·연락처·보존 정책·처리 위탁사를 채워 법률 검토해야 하는 제품 기준 초안입니다.

## 브라우저에 저장되는 정보

- 블로그 ID와 공식 주소
- 경험 답변·메모·사진 설명
- 원문을 제외한 말투 특징 카드
- 생성·편집한 초안과 최근 본문 버전 10개
- 재인코딩해 메타데이터를 제거한 취재 사진 최대 10장

사용자는 화면의 데이터 관리 기능에서 항목별 또는 전체 삭제할 수 있습니다. JSON 백업에는 취재 사진이 포함되지 않습니다.

## 서버에서 처리되는 정보

- OAuth 세션과 가명 처리된 네이버 회원 식별값: 메모리 세션 종료 또는 30분 만료 시 삭제
- AI 생성 요청의 글감·경험·말투·댓글: 요청 처리 중에만 사용하며 애플리케이션 서버에 저장하지 않음
- 감사 로그: 사건 종류·요청 ID·성공 여부만 기록하고 사용자 입력·IP·토큰·키는 기록하지 않음

OpenAI와 네이버 API를 활성화한 운영자는 각 처리 위탁·국외 이전·보존 정책을 실제 배포 지역과 계약에 맞게 별도로 고지해야 합니다.
