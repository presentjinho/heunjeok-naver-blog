# 개인정보 처리 기준

이 문서는 GitHub Pages 공개판의 데이터 처리 기준입니다. 공개판은 로그인·계정 연결·자동 발행·서버 AI 호출을 사용하지 않습니다.

## 브라우저에 저장되는 정보

- 경험 답변·메모·사진 설명
- 원문을 제외한 말투 특징 카드
- 생성·편집한 초안과 최근 본문 버전 10개
- 재인코딩해 메타데이터를 제거한 취재 사진 최대 10장

사용자는 화면의 데이터 관리 기능에서 항목별 또는 전체 삭제할 수 있습니다. JSON 백업에는 취재 사진이 포함되지 않습니다.

## 공개판에서 외부로 전송되는 정보

글 작성 데이터는 흔적 서버로 전송되지 않습니다. 사용자가 네이버·X·검색 링크를 누르면 해당 외부 사이트가 열리며 이후 처리는 각 사이트의 정책을 따릅니다. Google Fonts 요청은 실패해도 시스템 글꼴로 대체됩니다.

저장한 JSON 백업에는 작성 데이터가 들어 있으므로 본인 기기에 보관하고 공개 저장소에 올리지 마세요. 사진은 JSON 백업에 포함되지 않습니다.
